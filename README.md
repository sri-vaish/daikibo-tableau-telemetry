## Task Overview
This project analyzes telemetry data collected by Daikibo using Tableau.

## Steps Performed
- Imported JSON telemetry data into Tableau
- Created a calculated field **Unhealthy** assigning 10 minutes downtime for each unhealthy status
- Built bar chart: **Down Time per Factory**
- Built bar chart: **Down Time per Device Type**
- Created an interactive dashboard
- Enabled factory-level filtering on device-level downtime
